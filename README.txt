Space Invaders Game
-------------------

This is a simple Space Invaders game created in Python.

Requirements:
- Python 3.6 or newer

How to Run the Game:
1. Clone this repository.
2. Open a terminal and navigate to the cloned repository's directory.
3. Run the `run_space-invaders.sh` script by typing `run_space-invaders.sh` in the terminal. The game will start after the script completes.

Notes:
- The `run_space-invaders.sh` script will check for Python, create a virtual environment, install necessary dependencies, and run the game.
- Make sure the `run_space-invaders.sh` script is executable. If it is not, you can make it so by running the command `chmod +x run_space-invaders.sh` in the terminal.
